# Dash-Dost: Your Data, Unlocked.

Raw data is just noise until it’s visualized. Dash-Dost is the bridge between chaotic CSVs and professional, actionable dashboards—built by you, powered by AI.

Stop wrestling with complex BI tools or rigid spreadsheet templates. Dash-Dost uses a sophisticated data binding engine and intelligent layout generation to turn your raw files into actionable, insightful workspaces in seconds.

## The Dash-Dost Advantage

*   **Intelligent Binding Engine**: Stop fighting column mismatches. Our engine automatically resolves data fields to their intended charts, even when headers are cryptic, messy, or inconsistent.
*   **"Quick View" Mode**: Need insights *now*? Our zero-LLM mode instantly generates a functional, readable dashboard the moment you upload your file—no AI overhead, no wait time.
*   **AI-Driven Intent**: From KPI cards that calculate meaningful trends to complex time-series analysis, Dash-Dost understands the story your data is trying to tell.
*   **Adaptive Precision**: Every dashboard is built on a responsive grid, ensuring your insights are always clean, legible, and actionable, whether viewed on a desktop monitor or a tablet.

## Getting Started

### Prerequisites
*   Node.js (version 18 or above)
*   npm

### Installation

1.  **Clone & Setup**:
    ```bash
    # Clone the repo and install dependencies
    npm install
    ```

2.  **Configure Environment**:
    Duplicate `.env.example` as `.env` and set your API keys:
    ```env
    GEMINI_API_KEY="your-api-key-here"
    ```

3.  **Launch**:
    ```bash
    # Run the development server
    npm run dev
    ```
